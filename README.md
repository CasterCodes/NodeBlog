## Simple node appliaction

This is a simple node blog application. Single user

## To use this application

### Install all dependencies

```bash
   npm install
```
### Run the application
```bash
   npm run dev
```

### Dependencies
- express
- ejs
- express-ejs-layouts
- method-override
- jsdom
- dompurify
- marked
- slufigy
- mongo
- mongoose
